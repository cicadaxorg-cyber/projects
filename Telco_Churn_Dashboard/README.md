# Telco Customer Churn Analysis Dashboard

## Problem
Why do telecom customers leave?

## Key Findings
- Month-to-month contracts churn 3x more than annual contracts
- First 12 months are highest risk
- Contract type is strongest predictor

## Tools Used
- Power BI Desktop
- Power Query (data cleaning)
- DAX (calculations)

## Dashboard Features
- Churn rate KPI card
- Trend lines by tenure
- Segmentation by contract type
- Interactive filters

## How to Use
1. Download the .pbix file
2. Open in Power BI Desktop
3. Explore with slicers to filter by contract type and service
